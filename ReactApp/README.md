# Bus Wankers React Application

This is a React-based web application that displays the documentation for the Bus Wankers Glastonbury autofill functionality.

## Features

- Responsive design that works on desktop and mobile devices
- Displays instructions for using the autofill feature
- Shows images and video demonstrations
- Includes download links for the autofill CSV file

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Build for production:
   ```
   npm run build
   ```

## Files Structure

- `src/components/BusWankersPage.jsx` - Main component displaying the documentation
- `src/components/BusWankersPage.css` - Styling for the documentation page
- `src/App.jsx` - Main App component
- `src/index.js` - Entry point for the React application

## Usage

This application is designed to replace the static HTML documentation page (`wankers.html`) with a more modern, React-based interface that provides the same information in a more interactive and responsive way.
